# CodeAlpha_Phishing_Awareness
This repository contains a Phishing Awareness Training presentation created as part of a Cyber Security Internship. It explains different types of phishing attacks, how to identify them, and best practices to stay safe from cyber threats.
Training
📌 Overview

This project is focused on Phishing Awareness Training as part of a cybersecurity internship. The aim is to understand phishing attacks, how they work, and how users can protect themselves from becoming victims.

🎯 Objective

The main objectives of this task are:
To understand phishing attacks and their types
To identify phishing emails and fake websites
To learn social engineering techniques used by attackers
To follow best practices for online safety
To improve cybersecurity awareness
📧 What is Phishing?

Phishing is a type of cyber attack where attackers pretend to be trusted sources like banks, companies, or services to steal sensitive information such as passwords, OTPs, and bank details.
Phishing can occur through:
Email messages
Fake websites
SMS messages
Phone calls
⚠️ How to Identify Phishing Emails

Common signs of phishing emails include:
Unknown or suspicious sender address
Urgent or threatening language
Requests for personal or financial information
Spelling and grammar mistakes
Suspicious links or attachments
🌐 How to Identify Fake Websites

To identify fake websites:

Check the URL carefully for spelling errors
Look for secure connection (HTTPS)
Avoid clicking unknown or shortened links
Verify the authenticity of the website before entering data
🧠 Social Engineering Techniques

Attackers use psychological tricks such as:

Creating urgency (“Act immediately or account will be blocked”)
Creating fear (“Unauthorized login detected”)
Offering rewards (“You won a prize”)
Impersonating authority figures (banks, IT support)
Building trust by pretending to be familiar contacts
✅ Best Practices to Stay Safe
Never share passwords or OTPs
Enable Two-Factor Authentication (2FA)
Verify sender before clicking links
Avoid downloading unknown attachments
Keep software and browsers updated
Report suspicious messages
